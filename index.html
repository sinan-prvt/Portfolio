01. About Me

I’m Mohamed Sinan, a Full Stack Developer from Kerala, India, with hands-on experience building personal and academic web applications using Python, Django, Django REST Framework, React, and PostgreSQL.

I designed and developed full-stack projects to strengthen my understanding of backend architecture, REST API design, authentication flows, and database modeling. On the frontend, I built responsive user interfaces using React and Tailwind CSS, focusing on clean structure and usability.

My projects include an AI-inspired event management system and a full-stack eCommerce application, developed as self-driven projects to practice microservice concepts, containerization with Docker, asynchronous workflows, and secure API communication. These projects were not client-based but were built to simulate real-world systems and workflows.

I completed my higher secondary education at PKMMHSS Edarikode (2023–2025) and continue to deepen my knowledge in backend engineering, system design, and cloud-native development. I’m focused on building strong fundamentals and writing clean, maintainable code while preparing for real-world engineering challenges.
